#  TP 7 – Comprendre JSX, la composition avancée et les tests React

## Objectif du TP
Ce projet a pour but de pratiquer l’utilisation avancée de **JSX**, la **composition de composants React**, les **Higher-Order Components (HOC)**, les **Render Props**, ainsi que l’écriture de **tests unitaires** pour vérifier le comportement des composants.

---

## Concepts abordés
- Utilisation de **JSX** (JavaScript XML)
- Composition avancée des composants
- Création et utilisation de **Higher-Order Components (HOC)**
- Implémentation d’un **Render Props**
- Écriture de tests avec **React Testing Library** et **Jest**
